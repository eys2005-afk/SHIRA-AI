# סוכן יומן הדיונים - שירה ← Verbit

סוכן שמחליף את העבודה הידנית היומית: כל בוקר הוא נכנס ליומן הדיונים
במערכת שירה, מושך את תיקי אותו היום, וקובע לכל אחד מהם דיון (הקלטה)
ב-Verbit כולל הגורמים הקבועים. במהלך היום, מסך פשוט אחד ("הדשבורד")
מציג את כל דיוני היום עם כפתור גדול "התחל הקלטה" - כי הדיונים לא תמיד
מתחילים בשעה שנקבעה, וההפעלה בפועל נשארת בידיים שלך.

## שלבי הפרויקט

| שלב | מה קורה | סטטוס |
|-----|---------|-------|
| א' | ריצת בוקר אוטומטית: שירה ← Verbit | בנוי - דורש כיול חד-פעמי (למטה) |
| ב' | דשבורד להפעלת הקלטות בלחיצה | בנוי - תלוי בכיול של שלב א' |
| ג' | השוואת פרוטוקול מול תמלול ההקלטה | שלד מוכן (`agent/compare.py`) - לשלב מאוחר יותר |

## איך זה בנוי

הכול רץ **על מחשב העבודה שלך** (לא בענן) - כי שירה היא מערכת פנימית
שההזדהות אליה אוטומטית רק מהמחשב שלך. הסוכן פותח דפדפן (Edge/Chrome)
עם פרופיל קבוע משלו, כך שההזדהות האוטומטית לשירה וההתחברות ל-Verbit
נשמרות בין ריצות.

```
hearing-agent/
  config.yaml          לוח הבקרה: כתובות, selectors, גורמים קבועים
  agent/
    shira.py           קריאת יומן הדיונים משירה (+ מצב כיול)
    verbit.py          קביעת דיונים והפעלת הקלטות ב-Verbit (דפדפן או API)
    morning.py         ריצת הבוקר: שירה -> Verbit -> קובץ היום
    compare.py         שלב ג': השוואת פרוטוקול מול תמלול (Claude API)
    store.py           שמירת מצב היום ב-data/hearings-YYYY-MM-DD.json
  dashboard/           המסך הפשוט: דיוני היום + כפתורי הקלטה
  scripts/             קבצי הפעלה ל-Windows + רישום משימת בוקר מתוזמנת
```

## התקנה (פעם אחת, על מחשב העבודה)

דרישה מוקדמת: Python 3.11 ומעלה מותקן ([python.org](https://www.python.org/downloads/)),
עם סימון "Add Python to PATH" בהתקנה.

```bat
cd hearing-agent
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
playwright install msedge
copy .env.example .env
```

## שלב הכיול (פעם אחת - ובעזרתי)

לי (Claude) אין גישה לשירה ול-Verbit, ולכן שני מסכים צריכים "להצטלם"
פעם אחת כדי שאתאים אליהם את הסוכן:

1. **שירה:** הרץ `python -m agent.shira --calibrate`. ייפתח דפדפן -
   נווט בו ליומן הדיונים של היום, חזור לחלון השחור ולחץ Enter.
   נשמרים צילום מסך ו-HTML בתיקייה `data/calibration/`.
2. **Verbit:** הרץ `python -m agent.verbit --calibrate` ואותו תהליך -
   הגע למסך שבו קובעים דיון חדש.
3. שלח אליי את הקבצים מ-`data/calibration/`, ואני אמלא את כל שדות
   ה-selectors ואת הכתובות המדויקות ב-`config.yaml`.
4. עדכן ב-`config.yaml` גם את `fixed_participants` (הגורמים הקבועים)
   אם יש כאלה שאינם מופיעים ביומן עצמו.

בלי הכיול הזה הסוכן יעצור עם הודעה ברורה שמסבירה מה חסר - הוא לא
ינחש ולא יקבע דיונים שגויים.

## שימוש יומיומי

- **ריצת הבוקר** רצה אוטומטית אחרי שרושמים אותה פעם אחת:
  קליק ימני על `scripts/register_task.ps1` ← Run with PowerShell.
  (ברירת מחדל: ימים א'-ה' בשעה 07:30; המחשב צריך להיות דולק.)
  אפשר גם ידנית: `scripts\run_morning.bat`.
- **הדשבורד:** הפעל את `scripts\run_dashboard.bat` - נפתח דף בדפדפן
  עם דיוני היום. ליד כל דיון כפתור "▶ התחל הקלטה" (וכשההקלטה רצה -
  "■ עצור הקלטה"). כפתור "רענון מהיומן" מריץ את משיכת היומן שוב
  באמצע היום אם נוספו תיקים.

## Verbit: דפדפן או API?

כרגע `verbit.mode: browser` - הסוכן מפעיל את ממשק הווב של Verbit
בדיוק כפי שאתה היית עושה. זה עובד, אבל רגיש לשינויים בעיצוב האתר.

הדרך היציבה יותר היא ה-API של Verbit (`verbit.mode: api`). מאחר
שניסיון האינטגרציה הקודם לא צלח, המבנה מוכן בקוד אבל כבוי; כדי
להפעיל אותו צריך לקבל מ-Verbit (דרך איש הקשר של בית הדין) את כתובת
ה-API, מפתח (token), ותיעוד קצר - ואז נאמת יחד את הנתיבים המסומנים
`TODO` ב-`agent/verbit.py`.

## שלב ג': השוואת פרוטוקול מול תמלול

כשיהיו תמלולים אמיתיים, מריצים:

```bat
python -m agent.compare protocol.txt transcript.txt -o report.md
```

הדוח מפרט מה נאמר בדיון ולא נכנס לפרוטוקול, אי-התאמות, והצעות שיפור.
דורש מפתח `ANTHROPIC_API_KEY` ב-`.env` ו-`pip install anthropic`.

## אבטחה ופרטיות

- שום סיסמה לא נשמרת בקוד או בהגדרות; ההזדהות נעשית דרך פרופילי
  הדפדפן המקומיים בלבד.
- הדשבורד מאזין רק ל-`127.0.0.1` - נגיש מהמחשב שלך בלבד.
- נתוני הדיונים נשמרים מקומית בלבד (`data/`, מוחרג מ-git).
- בשלב ג' תוכן הפרוטוקול והתמלול נשלח ל-Claude API - יש לוודא שזה
  מותר לפי כללי אבטחת המידע של המערכת לפני השימוש.
